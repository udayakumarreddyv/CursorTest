## Employee API (Spring Boot 2.7, Java 1.8)

Run locally:

```bash
mvn spring-boot:run
```

API endpoints:
- GET `/api/employees`
- GET `/api/employees/{id}`
- POST `/api/employees`
- PUT `/api/employees/{id}`
- DELETE `/api/employees/{id}`

H2 console at `/h2-console` (JDBC URL: `jdbc:h2:mem:employees`).
